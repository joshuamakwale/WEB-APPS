# Animated To-Do List (JavaScript)

## Overview

This project is a simple animated To-Do List web application built using **HTML, CSS, and JavaScript**.
It allows users to add tasks with a date and time, display them on the page, and remove tasks with animated transitions.

The project was created as part of learning **frontend development and JavaScript DOM manipulation** before integrating backend frameworks such as **Flask**.

---

## Features

* Add a task with **text, date, and time**
* **Animated task cards** when tasks appear
* **Delete animation** when removing tasks
* Interactive **hover effects**
* Input focus animations
* Clean and simple UI design

---

## Technologies Used

* **HTML5** – page structure
* **CSS3** – styling and animations
* **JavaScript** – DOM manipulation and interactivity

---

## Purpose of the Project

This project was built to practice:

* JavaScript event handling
* DOM manipulation
* UI animations with CSS
* Basic frontend project structure

It is also intended as preparation for integrating a **Flask backend** where tasks could later be stored in a database.

---

## How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in a web browser

No installation or dependencies are required.

---
